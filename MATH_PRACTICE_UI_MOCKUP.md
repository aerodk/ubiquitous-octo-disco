## Colors

- **Success**: Green (#4CAF50)
- **Error**: Red (#F44336)
- **Info**: Blue (#2196F3)
- **Hint**: Amber/Yellow (#FFC107)
- **Background**: Light amber for hint box (rgba(255, 193, 7, 0.1))
- **Border**: Amber (#FFC107)

## Typography

- **Problem Text**: 48px, bold
- **Operation Label**: 16px, gray
- **Answer Input**: 24px, centered
- **Feedback**: 18px, bold
- **Hint**: 16px, regular
- **Statistics**: 16px, bold

## Interaction Flow

1. User clicks calculator icon (ğŸ”¢) on setup screen
2. Math Practice screen opens
3. Random problem appears (or from selected type)
4. User types answer
5. User presses Enter or clicks "Tjek Svar"
6. Feedback appears based on correctness
7. If wrong twice, hint appears
8. If correct or skipped, auto-advance after 2 seconds
9. Repeat from step 3

## Accessibility

- Large touch targets (buttons minimum 50px height)
- Clear color contrasts
- Number-only keyboard automatically shown
- Large text for readability
- Clear visual feedback for all actions
- Danish language throughout
