# User Guide: Large Screen Mode

## What is Large Screen Mode?

Large Screen Mode is a feature that makes the app easier to view on big screens when you're using a mobile device. When you connect your phone or tablet to a TV, projector, or monitor, the normal mobile layout can appear too small. Large Screen Mode scales up all the text, buttons, and cards so everyone can see clearly.

- üë• Showing standings to a group of people
- üéÆ Using your device with a larger external display

## How to Use Large Screen Mode

### Activating Large Screen Mode

1. Open the tournament app on any of these screens:
   - **Leaderboard** (standings screen)
   - **Round Display** (lane view with matches)
   - **Tournament Completion** (final results)

2. Look for the icon in the top-right corner of the screen:
   - üì± **Phone icon** = Currently in Mobile Mode
   - üñ•Ô∏è **Desktop icon** = Currently in Desktop Mode

3. Tap the icon to switch modes:
   - Tapping üì± switches to Desktop Mode (larger layout)
   - Tapping üñ•Ô∏è switches back to Mobile Mode (normal layout)

When you switch to Desktop Mode:

## Troubleshooting

### The text is still too small on my TV

1. Make sure you've tapped the desktop icon (should show üñ•Ô∏è when active)
2. Check your TV's display settings - some TVs have zoom/picture size options
3. Try both portrait and landscape orientations
4. Consider your viewing distance - Desktop Mode is optimized for 2-3 meters viewing distance

### The text is too large on my phone

1. Tap the üì± phone icon to switch back to Mobile Mode
2. The app remembers your last choice, so it will stay in Mobile Mode next time

### The layout looks weird

1. Try rotating your device (portrait/landscape)
2. Toggle between modes to find which looks better
3. Desktop Mode is designed for screens 24" and larger - smaller screens may look crowded

### The toggle doesn't appear

- ‚úÖ Leaderboard Screen

## Frequently Asked Questions

**Q: Does it affect the tournament data?**  
A: No, it only changes how things are displayed. Your tournament data is safe.

**Q: Can I use different modes on different screens?**  
A: Currently, the mode applies to all screens. Whatever you choose on one screen applies everywhere.

**Q: Will it work on my TV?**  
A: Yes! As long as you can connect your device to your TV (HDMI, wireless mirroring, etc.), Desktop Mode will make everything easier to see.

**Q: Does it use more battery?**  
A: No, the display mode doesn't significantly affect battery usage.

**Q: Can I customize the scale amount?**  
A: Not in the current version. The app uses optimized scale factors (1.3x for fonts, 1.5x for elements).
