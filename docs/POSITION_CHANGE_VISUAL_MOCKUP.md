### Calculation Example
```
Previous Round (Round 3): Player was rank 5
Current Round (Round 4):  Player is now rank 3

Calculation: rankChange = previousRank - currentRank
                        = 5 - 3
                        = +2

Display: Green badge with "+2"
```

## Accessibility Notes

The feature uses both color AND text to convey information:
- Green badge shows `+` prefix
- Red badge shows `-` number
- Black badge shows `Â±` symbol
- Number value provides magnitude

This ensures users who are colorblind can still understand the change direction and magnitude.
