# Storage, Self-Registration & Sharing 
Version 8 is focused in integrating cloud storage and sharing of tournaments

## Highlights 
* Google Firebase 
* Name of tournament, create in advance 
* Saving tournament results
* Share code and enter player from everywhere 


## Feature F-025: Tournament Sharing
- Generer unik share code (6-character)
- Generér 6 optional cifret pass code 
- Share link: https://app.com/join/ABC123
- Copy link, QR code, direkte deling
- Public landing page med tournament info

## Feature F-026: Self-Registration
- Spillere åbner link
- Ser turnering detaljer
- "Tilmeld dig" knap
- Indtast navn (+ valgfri avatar upload)
- Bekræft deltagelse
- Tilføjet til live liste

## Feature F-027: Organizer Dashboard
- Se tilmeldte live
- Accept/reject spillere (optional)
- Luk tilmelding når fuldt
- Send notifikationer til deltagere

## Feature F-028: Optional Facebook Login
- Alternative login metode
- Hent navn + profilbillede automatisk
- Hurtigere for spillere med Facebook
