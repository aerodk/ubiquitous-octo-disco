## Toggle Button Behavior

**When Detailed View is Active:**
- Icon shows: ğŸ“‹ (view_compact)
- Tooltip: "Compact View"
- Click: Switches to compact view

**When Compact View is Active:**
- Icon shows: ğŸ“ (view_list)
- Tooltip: "Detailed View"
- Click: Switches to detailed view

## Long Press Interaction

**Works in Both Views:**
1. User long presses on any player card
2. Dialog appears with player's game history
3. Shows all matches that player participated in
4. Each match shows:
   - Round number
   - Partner
   - Opponents
   - Score
   - Result (color-coded: green for win, red for loss)

## Color Coding (Preserved in Both Views)

**Top 3 Positions:**
- 1st Place: Gold/Amber background, white text
- 2nd Place: Silver/Grey background, white text
- 3rd Place: Bronze/Brown background, white text
- Other: White background, dark text

## Benefits

**Compact View:**
- See more players at once (5-6 instead of 2-3)
- Quick overview of standings
- Easier to compare rankings
- Less scrolling needed

**Game History:**
- Understand individual performance
- See partnership patterns
- Review specific match results
- Track performance across rounds
